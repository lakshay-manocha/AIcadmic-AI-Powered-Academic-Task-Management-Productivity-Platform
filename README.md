# AIcadmic 
**AI-Powered Academic Task Management & Productivity Platform**

AIcadmic is a full-stack AI-driven academic planner designed to help students and professionals manage tasks efficiently, prioritize work intelligently, and improve overall productivity using Artificial Intelligence and Machine Learning.

---

##  Project Overview

Managing academic and professional tasks manually often leads to poor prioritization, missed deadlines, and reduced productivity. **AIcadmic** addresses this challenge by integrating AI-based task recommendations and predictions into a modern, scalable web application.

The platform combines a **Spring Boot backend**, **React frontend**, and **Machine Learning models** to deliver a smart, user-centric task management experience.

---

##  Key Features

-  AI-based task recommendation system  
-  Task priority prediction using ML  
-  Smart daily planner & productivity insights  
-  Secure authentication & authorization  
-  RESTful API architecture  
-  Scalable and modular three-layer backend design  
-  Modern, responsive UI with React

##  System Architecture

AIcadmic follows a **three-layer architecture**:

1. **Presentation Layer** – React-based frontend  
2. **Business Logic Layer** – Spring Boot services and AI integration  
3. **Data Layer** – MySQL database and ML models  

This ensures better scalability, maintainability, and security.
